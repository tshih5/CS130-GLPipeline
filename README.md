# CS130-GLPipeline
Rendering Pipeline for CS130
## Scripts

Available scripts:

### `scons`

Compiles rendering pipeline source files

### `./driver -i ./N.txt`

Runs test N (00-25)

### `./driver -i ./N.txt -s ./N.png`

Compares test N (00-25) with expected result and returns a diff
